To compile and run the C++ code...:
1. Put the appropriate C++ and header files into a directory, on a system running linux.
2. Enter: 'g++ -pthread -o trial avlTree.cpp bstree.hpp'
3. This should make a executable file. run it with './trial' This should produce an output file. Then, you're done! 


To compile and run the Go code...:
1. Install Google Go, and put the appropriate Go files into a directory. 
2. Make sure your gopath is set up correctly.
3. enter 'go run AVLTree.go', which will then run and produce an output file. Then, you're done!
